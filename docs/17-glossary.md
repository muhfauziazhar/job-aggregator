## Purpose

Single home for product- and project-specific terminology. Avoids the situation where different docs use the same word to mean different things, or use different words to mean the same thing.

Add a term when:
- It appears in 2+ docs without definition.
- A new team member would have to ask what it means.
- It carries domain-specific meaning that differs from common usage.
